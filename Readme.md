# 🚦🌆 City Traffic Flow Simulator

🚀 *Control the flow. Reduce congestion. Build smarter cities.*
An interactive web-based simulation of **urban traffic management systems** using **HTML, CSS & JavaScript**

---

## 🌟 Project Highlights

🚗 **Dynamic Vehicle Simulation**
→ Cars move continuously across lanes in real-time

🚦 **Traffic Signal Control**
→ Switch between **RED 🔴 / GREEN 🟢** manually

📊 **Live Traffic Metrics**
→ Monitor:

* Total cars 🚗
* Traffic flow rate ⚡
* Signal state 🚦

⚡ **Real-Time Simulation Engine**
→ Continuous updates using JavaScript loops

🎮 **Interactive Controls**
→ Add vehicles & control signals dynamically

## 🎮 How It Works

🚗 Cars are generated dynamically
➡️ Each car has:

* Position
* Speed
* Lane

⚙️ Simulation loop updates movement in real-time

🚦 Signal logic:

* RED → Cars stop near signal
* GREEN → Cars move freely
